# Join a server with an invite

Servers are private — you join one with an **invite link** or **invite code** that someone shares with you.

## With a link

Open the invite link (it looks like `…/invite/AbC123`). You'll see a **preview** of the server — its name, icon, and member count — then click **Join**.

![invite preview page with the Join button](assets/join-invite-1.png)

## With a code

Already signed in? Click the **+** to add a server, choose **Join with invite**, paste the code, and confirm.

![the Add Server modal, "Join" tab, with the code field](assets/join-invite-2.png)

## Make your own invite

Inside a server you manage, open **Server Settings → Invites** to create a code. You can set an **expiry** and a **maximum number of uses**, then copy the link to share.

![the Invites settings panel with an active invite and its options](assets/join-invite-3.png)
