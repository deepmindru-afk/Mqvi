# Pinned messages

Keep important messages easy to find.

## Pin a message

Hover a message and choose **Pin** (requires permission in servers). Pinned messages are collected in the channel's **pinned list**, so announcements, links, or rules don't scroll away.

> 📸 **Screenshot:** the pin action on a message.

## View the pins

Open the **pinned messages** panel for a channel to see everything that's been pinned, newest first. Unpin from there when something's no longer needed.
