# Sending messages

Type in the box at the bottom of any channel and press **Enter** to send. **Shift+Enter** adds a new line.

## Formatting

Messages support Markdown:

- `**bold**` → **bold**
- `*italic*` → *italic*
- `` `code` `` → inline `code`
- Triple backticks for code blocks

## Beyond plain text

- **Reply** — hover a message and choose reply to quote it
- **React** — add emoji reactions
- **Edit / delete** — change or remove your own messages
- **Mention** — type `@` to mention a person or role
- **Attach** — drag a file in, or paste an image straight from your clipboard
- **Pin** — keep important messages in the channel's pinned list

> Links automatically show a preview card when the site supports it.
