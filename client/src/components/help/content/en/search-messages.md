# Searching messages

Find that message you know you saw somewhere.

## Search a channel

Open search (or press **Ctrl/Cmd + F**) and type what you're looking for. Results are listed with context, and you can page through them to jump straight to a message.

> 📸 **Screenshot:** `assets/search-messages-1.webp` — the search box with a query and matching results.

## How it stays private

On normal servers, search runs on the server for speed. On **end-to-end encrypted** servers, the server can't read your messages — so search runs **locally on your device** against messages you've already received. Either way, your search query stays yours.
