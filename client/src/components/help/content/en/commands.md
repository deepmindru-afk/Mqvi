# Slash commands

Type **`/`** at the start of the composer to open the command menu — an autocomplete list shows what's available and how to use it as you type.

> 📸 **Screenshot:** the command autocomplete list after typing "/".

## Everyday

| Command | What it does |
| --- | --- |
| `/help` | List every command |
| `/search query` | Search the current channel |
| `/me action` | Send an action message — e.g. `/me waves` |
| `/shrug` | Append a shrug face to your message |
| `/poll "Question" "Option A" "Option B"` | Start a quick poll |

## Status & voice

| Command | What it does |
| --- | --- |
| `/status online`, `/status idle`, `/status dnd` | Change your presence |
| `/mute` | Toggle your microphone |
| `/deafen` | Toggle deafen |

## People

| Command | What it does |
| --- | --- |
| `/dm @user` | Open a direct message |
| `/call @user` | Start a 1-on-1 call |
| `/invite @user` | Invite someone to the server |
