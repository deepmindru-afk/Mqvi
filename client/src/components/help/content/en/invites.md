# Inviting people

## Create an invite

In **Server Settings → Invites**, create an invite **code**. Set an **expiry** and a **maximum number of uses**, then **copy the link** to share anywhere.

> 📸 **Screenshot:** `assets/invites-1.webp` — the Invites panel with an active invite and its expiry/uses options.

## Invite friends directly

Use **Invite Friends** to pick people from your friends list and send them an invite in one go.

> 📸 **Screenshot:** `assets/invites-2.webp` — the Invite Friends modal with a friend list and Send.

*(Joining a server with an invite is covered in **Getting Started**.)*
