## v2.6.0

- **auth**: add password visibility toggle on login and register forms

- **security**: scope typing broadcast to server members only, add HTTP security headers (X-Content-Type-Options, X-Frame-Options, Referrer-Policy, Permissions-Policy), add IP-based rate limiting to register/forgot-password/reset-password endpoints
- **voice**: add global PTT shortcut in Electron and focus-only warning on web

- **frontend**: extract shared message utilities (encryptFilesForE2EE, handleRateLimitError, createTypingHandler, updateMessageInRecord, deleteMessageFromRecord, updateReactionInRecord, updateAuthorInRecord) eliminating ~130 LOC duplication between messageStore and dmStore
- **frontend**: split useWebSocket routeEvent (1,256 LOC) into 4 domain handler modules — channelEventHandlers, dmEventHandlers, voiceEventHandlers, systemEventHandlers — reducing useWebSocket.ts to 383 LOC
