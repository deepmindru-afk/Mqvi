## v2.15.0

### New Features
- **voice**: ephemeral chat tied to the active voice channel session
- **shortcuts**: allow single-key mute/deafen bindings (no Ctrl/Alt required)
- **voice**: separate notification and app sound volume sliders
- **voice**: double-click screen share to toggle fullscreen
- **sidebar**: clicking a friend opens the DM directly instead of the friends tab
- **electron**: open external links in the default browser instead of a new window

### Bug Fixes
- **ui**: restore full-width admin pages
- **voice**: compute call duration each render to avoid a stale time jump
