## v2.16.0

### New Features
- **Calls**: 1-on-1 calls now fall back to a TURN relay when a direct connection can't be established, so they work behind restrictive networks/NATs — with automatic recovery (ICE restart) after a brief network drop.
- **Calls**: Fullscreen and per-stream volume control (0–200%) for 1-on-1 call video — right-click for the volume menu, double-click or the hover button for fullscreen.
- **Calls**: WhatsApp-style camera swap — when both people have their camera on, tap the floating self-view to swap it with the main view.
- **Voice**: Hovering a server now shows a popup of its active voice channels and who's currently in them.
- **Voice**: The Friends view now shows which friends are in a voice channel (and on which server) in the right panel.

### Bug Fixes
- **Voice**: No more spurious error toasts during transient voice reconnects — a single notice appears only if you're actually disconnected.
- **Calls**: Copying an image now works in the desktop app (native clipboard).
- **Feedback**: Reply attachments now upload correctly (restored a missing database column).
- **Admin**: Platform log entries expand on click to show the full message.
- **Sidebar**: A friend's status shows correctly on their avatar with live presence updates.
