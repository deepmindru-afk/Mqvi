### New Features

- **files**: per-type upload directory layout with centralized path validation and segment security
- **files**: HMAC-signed time-limited URLs for all file access with key rotation support
- **files**: permission-aware ACL on URL refresh — kicked users cannot extend expired URLs
- **files**: MIME-based safe-serve — inline display for images/video/audio, force-download for HTML/SVG/executables, max file size raised to 500MB
- **storage**: per-user upload quota with automatic disk cleanup on message/DM/soundboard delete
- **files**: one-time migration script to move legacy uploads to the new segregated layout
